Productum

Приложение для распознавания состава продуктов

Demo

Что такое Productum?

Productum - программа, которая распознает состав продукта и выявляет вредные добавки и аллергены.

Здоровое питание

Приложение может выявлять вредные для организма добавки

Экономия времени

Приложение позволяет не тратить время на запоминание вредных добавок и чтение состава продукта

Программа подстраивается под ваши аллергии

Пользователь может сам выбрать какие продукты он не может употреблять в пищу

Мобильность

Приложение запускается на всех устройствах, где есть браузер

Бесплатность

Приложение распространяется полностью бесплатно

Простота

Приложение берет всю работу на себя
